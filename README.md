# PHP Integration Layer

This project provides the integration logic for submitting verified purchase orders to the database. It handles PDF extraction and order processing.

## Features
- PDF data extraction
- Order verification and submission
- Database interaction logic

## Getting Started
To use this integration layer, ensure you have a local PHP environment (like XAMPP) running.
